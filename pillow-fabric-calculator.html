How Pillow Yardage Is Calculated

Each pillow requires fabric for a front and back piece, plus seam allowances. The formula accounts for:

  • Pillow face: Length + seam x 2, Width + seam x 2 (front piece)
  • Pillow back: Same as front, or 2 overlapping pieces for an envelope closure
  • Flange: Adds the flange width x 2 to both dimensions
  • Fabric width: Determines how many pieces fit side by side
