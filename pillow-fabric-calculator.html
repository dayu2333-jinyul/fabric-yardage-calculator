How This Pillow Fabric Calculator Works

This pillow fabric calculator accounts for front and back pieces, seam allowances, flange width, and fabric width all at once. Unlike a generic yardage estimator, this pillow fabric calculator is built specifically for cushions — it knows that each pillow needs two pieces (front + back), that envelope closures need extra overlap fabric, and that flange borders add to dimensions on all four sides.

The pillow yardage estimator behind this tool uses five inputs: pillow dimensions, number of pillows, fabric width, seam allowance, and waste percentage. Together, they give you exact cushion yardage in yards, inches, centimeters, and meters. Select one of the standard sizes above (16x16 throw, 18x18 euro, 12x20 lumbar) or enter custom dimensions.

  • Pillow face: Length + seam × 2, Width + seam × 2
  • Pillow back: Same as front, or 2 overlapping pieces for envelope closure
  • Flange: Adds flange width × 2 to both dimensions
  • Fabric width: Determines how many pillow pieces fit side by side per row

Pillow Styles — What This Pillow Fabric Calculator Handles

This pillow fabric calculator handles every common pillow style. The cushion yardage changes significantly based on the style you choose, so select the right one before running the calculator.

Knife-Edge Pillows

The most efficient style for a pillow fabric calculator: two pieces, straight seams. A 16×16" knife-edge pillow on 54" fabric needs about 5/8 yard according to this pillow yardage estimator — both pieces fit side by side within a single width. This is the minimum cushion yardage you can achieve.

Box Cushions and Seat Cushions

Box cushions need top, bottom, and a boxing strip. An 18×18" box cushion (2" thick) on 54" fabric: the pillow fabric calculator shows 3/4 yard for top + bottom, plus separate boxing. Total: 1–1.5 yards. Use the cushion fabric calculator for exact boxing strip yardage scenarios.

Pillows With Welt Cord or Piping

Adding piping increases your pillow fabric calculator result by about 1/8 yard for the bias strip. The pillow yardage estimator cannot guess your piping fabric — treat it as a separate material purchase.

Envelope Closure vs. Zipper vs. Hidden Zipper

  • Envelope closure: Back has 2 overlapping pieces. Add 4" to back length. Budget 5-10% extra cushion yardage.
  • Standard zipper: Same yardage as knife-edge. No extra fabric. Add to the seam allowance in the pillow fabric calculator.
  • Hidden/invisible zipper: Same as standard — the pillow yardage estimator does not change.

Batch Sewing — Optimizing Pillow Yardage

Run this pillow fabric calculator for multiple pillows and you will see diminishing per-pillow yardage. Two 18×18" pillows on 54" fabric: 4 pieces all fit in one fabric width (54" ÷ 19" = 2 per row, 2 rows = 38" total). That is just over 1 yard for 2 pillows — barely more than 1 pillow alone.

Pillow Fabric Selection — How It Affects Calculator Results

The fabric you choose changes the pillow yardage. Here is how each type interacts with this pillow fabric calculator:

  • Cotton / quilting (45"): Needs more cushion yardage due to narrower width. The pillow fabric calculator will show you the difference compared to 54".
  • Velvet (54"): Napped — all pieces face the same direction. The pillow yardage estimator cannot auto-detect this; manually add 15% to the result.
  • Upholstery fabric (54-60"): Pattern matching for large prints adds 20-30%. Run the pillow fabric calculator without pattern match, then add 1/4–1/2 yard.

Frequently Asked Questions

How much fabric do I need for an 18x18 throw pillow?
An 18x18 throw pillow needs about 1/2 yard of 54-inch fabric or 3/4 yard of 45-inch fabric. The pillow fabric calculator above accounts for front and back pieces, seam allowance, and fabric width automatically.
Can I use this pillow fabric calculator for box cushions?
This calculator works best for knife-edge pillows. For box cushions with a boxing strip (side panel), use the cushion fabric calculator for accurate yardage including the separate boxing piece.
Should I add extra fabric for pattern matching on pillows?
Yes, add 20 to 30 percent extra yardage for pattern matching on large prints or stripes. The pillow fabric calculator lets you set a waste percentage — increase it to account for pattern alignment.
How much fabric for a 16x16 inch pillow?
A 16-inch square pillow with front and back pieces on 54-inch fabric needs about 1/2 yard. That covers both pieces side by side across the width plus seam allowance.
Do I need more fabric for a pillow with a flange?
Yes. A flange adds 2-4 inches per side to each piece. A 2-inch flange on an 18-inch pillow requires cutting 22-inch pieces. Adjust piece dimensions in the calculator accordingly.
