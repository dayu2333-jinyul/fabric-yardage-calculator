How to Use This Curtain Fabric Calculator in Inches

All fabric yardage for curtains starts with inches — whether you are using a tape measure or reading a window size chart. This calculator keeps everything in inches so there is no confusion converting between feet and yards prematurely.

  1. Measure window width in inches — from left edge to right edge of the window frame.
  2. Measure window drop in inches — from the top of the rod to where you want the curtain to end (sill, below sill, or floor).
  3. Choose fullness — 2x means the fabric is twice as wide as the window for standard gathers.
  4. Select fabric width — standard decor fabric is 54 inches wide. Wide fabrics at 60" and extra-wide at 108" reduce the number of seams.
Inches to yards reminder: 36 inches = 1 yard. The calculator handles the conversion — your result is shown in yards with a metric equivalent.
